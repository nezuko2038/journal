# MacにリモートログインでSSHログインする



## Macでリモートログインを有効にする

* デスクトップ左上のりんごマークをクリック
* `System Preference` -> `Sharing`
* Remote Login にチェックを入力し、リモートログインを有効にします。


## SSHログイン

あとは、ターミナルでログイン可能か確認します。


```
ssh 192.168.XX.XX
```
